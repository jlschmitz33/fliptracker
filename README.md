# FlipTracker

Track your boat and outboard motor flipping business — profits, expenses, repairs, and sales.

---

## Deploy to Vercel (5 steps)

### Step 1 — Create a Turso database (free)

```bash
# Install Turso CLI
winget install turso    # Windows
# or: curl -sSfL https://get.tur.so/install.sh | bash   (Mac/Linux)

# Login
turso auth login

# Create a database
turso db create fliptracker

# Get your URL and token (save these)
turso db show --url fliptracker
turso db tokens create fliptracker
```

### Step 2 — Initialize the database schema

After you have the URL and token, add them to `.env.local`:
```
cp .env.local.example .env.local
# edit .env.local with your Turso credentials
```

Then run locally once to create tables:
```bash
npm run dev
# Visit: http://localhost:3000/api/init  (POST request, or use the button on dashboard)
```

### Step 3 — Push to GitHub

```bash
git init
git add .
git commit -m "Initial FlipTracker"
git remote add origin https://github.com/yourusername/fliptracker.git
git push -u origin main
```

### Step 4 — Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
2. Add environment variables in Vercel dashboard → Settings → Environment Variables:
   - `TURSO_DATABASE_URL` = your Turso URL from step 1
   - `TURSO_AUTH_TOKEN` = your Turso token from step 1
3. Click Deploy

### Step 5 — Enable photo uploads (Vercel Blob)

In your Vercel project dashboard:
1. Go to **Storage** → **Create Database** → **Blob**
2. Connect it to your project
3. The `BLOB_READ_WRITE_TOKEN` is added automatically

Then visit your deployed app and load sample data from the dashboard.

## Local Development

```bash
npm install
cp .env.local.example .env.local
# Add your Turso credentials to .env.local

npm run dev
# Open http://localhost:3000
```

**First time setup:** After starting, call `POST /api/init` to create DB tables:
```bash
curl -X POST http://localhost:3000/api/init
```

Or click the "Initialize DB" button that appears if no tables exist.

## Features

| Feature | Details |
|---------|---------|
| **Dashboard** | Total profit, avg ROI, best/worst flip, monthly chart |
| **Inventory** | Search + filter by status/type, progress bars |
| **Item Detail** | Inline-edit all fields, no separate edit page needed |
| **Expenses** | Quick-add with category breakdown, delete individual |
| **Repair Checklist** | Auto-generated by type, add custom tasks, toggle done |
| **Sale Tracking** | Listed platforms (toggle), buyer info, profit breakdown |
| **Photo Upload** | Vercel Blob in production, local folder in dev |
| **CSV Export** | `/api/export` or "Export CSV" in nav |

## Tech Stack

- **Next.js 16** (App Router, TypeScript)
- **Turso** — serverless SQLite (free tier: 9GB, 1B row reads/month)
- **Vercel Blob** — photo storage (free tier: 1GB)
- **Tailwind CSS** + **Recharts** + **lucide-react**
